## 新武器设计：底部远程武器

### 设计理念

与现有"手指在哪武器在哪"的近战模式不同，机关枪和火箭弹属于**底部远程武器**：

- **位置约束**：Y轴固定在防线后方，只有X轴跟随手指横向滑动
- **自动射击**：不依赖点击/滑动触发，按固定频率自动开火
- **始终可见**：无论手指是否触摸屏幕，武器始终渲染（像炮台一样陈列在底部）
- **子弹实体**：发射的子弹/火箭是独立实体，飞行、碰撞、消失

定位：高射速、低伤害、持续压制

#### 特殊能力：加特林

- 费用：5000💰
- 效果：射速翻倍（14发/秒），子弹穿透敌人
- 描述：子弹穿透小猪继续飞行，一串全中

#### 子弹参数

- 碰撞半径：6px（小圆点）
- 飞行方向：垂直向上
- 击退力：3（微弱，靠数量取胜）
- 同屏最多子弹数：约12发（14rps × 0.8s飞行时间）

#### 视觉设计

- 枪身：军绿色长方形，带握把和弹匣
- 枪管：略浅绿色，向上延伸
- 枪口火焰：每次射击时短促的金色/白色闪光
- 子弹：金色小圆点 + 拖尾短线
- 后坐力：射击时枪身微小上弹动画

#### 特殊能力：双管火箭

- 费用：8000💰
- 效果：一次发射两枚火箭（并排，间距20px）
- 描述：双发齐射，爆炸覆盖更广

#### 燃烧效果

- 持续时间：3秒
- 每秒伤害：10（基础）
- 效果：被爆炸波及的小猪持续燃烧掉血

#### 爆炸行为

- 触发条件：碰到小猪 OR 飞到屏幕35%高度（约半屏）
- 爆炸持续：400ms（视觉+碰撞判定窗口）
- 击退力：25（击飞效果），方向强制向上
- 同屏最多火箭数：约2-4发

#### 视觉设计

- 发射器：暗红色炮管，粗壮
- 飞行火箭：红色弹体 + 金色弹头 + 红色尾翼 + 火焰尾焰粒子
- 爆炸：三层同心圆（红→橙→金），带粒子扩散
- 冷却指示：炮口微弱的烟雾

### 与现有武器的对比

| 维度 | 近战武器 | 底部远程武器 |
|------|---------|------------|
| 位置 | 跟随手指自由移动 | Y固定底部，X跟手指 |
| 触发方式 | 滑动/点击触发 | 自动按频率射击 |
| 渲染 | 仅触摸时显示 | 始终显示 |
| 打击感 | 碰撞即反馈 | 子弹飞行→命中延迟反馈 |
| 策略 | 主动追击目标 | 走位拦截，预判射击线 |
| 适合场景 | 猪分散时 | 猪集中成列时 |

### 操作体验

1. 手指横向滑动 → 武器沿底部左右移动
2. 武器自动向上射击，无需额外操作
3. 玩家策略从"追着猪打"变为"预判猪的路径，提前走位拦截"
4. 机关枪适合扫射宽度，火箭弹适合炸密集群
5. 手指离开屏幕 → 武器停在上次位置，继续自动射击
