# 《暴打小猪》V2.0 PRD — "给你的烦恼取个名字，然后揍飞它"

---

## 定位

**不是打地鼠。是给"不爽"取个名字，然后亲手解决。**

同类游戏（消灭病毒、水果忍者、打地鼠类）的敌人都是无脸怪。我们的核心差异：**每一只小猪都有名字、有性格、有台词。** 玩家不是在杀怪，是在发泄——打老板、打前任、打昨天的自己。

目标用户：微信生态下的休闲解压玩家。核心情绪驱动：**"我把 XXX 揍飞了 100 次"是天然的情绪出口和社交货币。**

---

## 微信能力边界

| 能力 | 可做？ | 说明 |
|------|--------|------|
| 获取玩家自己头像/昵称 | ✅ | `wx.createUserInfoButton` 主动授权 |
| 获取微信好友头像/昵称 | ❌ | 隐私政策限制，开放数据域只能渲染不能导出 |
| 获取群排行数据 | ✅ | `shareTicket` + `wx.getGroupCloudStorage`，仅限游戏分数 |
| 分享卡片到群 | ✅ | `wx.shareFileMessage` / `wx.shareAppMessage` |
| 通过分享链接传递数据 | ✅ | `query` 参数可携带少量数据 |
| 玩家自定义昵称 | ✅ | 已有 `Storage.setNickname()` |

## Phase 1：猪的人格化（已有资产深化）

### 1.1 小猪身份系统（务实方案）

不依赖微信好友导入，改为**玩家自主管理小猪身份池**：

**身份来源**（按优先级）：
1. **自定义名单**（新功能）：玩家在设置里维护一个"欠揍名单"，输入名字（最多 20 个），关卡中的猪随机从名单取名。名字旁显示该名字被揍的总次数。
2. **玩家自己**（已有）：玩家的微信头像/昵称作为小猪身份来源。菜单页玩家信息卡旁边放一个开关："把我自己放入猪池"——打"自己"有额外金币奖励。
3. **emoji 库 + 装扮**（已有）：小猪外观来自 emoji + ShopConfig.pigOutfits 装扮叠加。

**"欠揍名单"UI**：
- 菜单页新增入口：`📋 欠揍名单`
- 列表形式展示所有名字，每个名字后面显示被揍次数
- "+"按钮添加新名字（键盘输入，最多 8 个中文字）
- 左滑删除（或长按）
- 建议占位文案："输入一个让你不爽的名字..."

**分享裂变**：通关后的分享卡片上，显示"我把 [名单中的名字] 揍飞了 XX 次"，接收方点击进入游戏后，这个名字自动加入接收方的欠揍名单候选。

### 1.2 小猪的"欠揍感"台词系统

**现状**：`DialogConfig.js` 有 5 个情绪池（arrogant/begging/broken/ko/boss），台词通用
**差距**：台词没有利用玩家数据和上下文

**改进**：

| 情绪 | 现有条数 | 扩充到 | 特点 |
|------|---------|--------|------|
| arrogant（嚣张） | 10 | 20 | 加入名字占位符 `{name}` |
| begging（求饶） | 10 | 20 | 同上 |
| broken（崩溃） | 10 | 15 | 同上 |
| ko（击飞） | 5 | 10 | 同上 |
| bossArrogant | 5 | 10 | 引用玩家数据 |
| bossDefeated | 3 | 5 | 同上 |

**和玩家数据联动**：
- `{name}` → 该猪的昵称（来自欠揍名单/玩家昵称）
- `{gold}` → 该猪携带的金币数
- `{combo}` → 当前玩家连击数
- `{weapon}` → 当前武器名称

**示例台词**：
- 嚣张时："`{name}` 你都打不过，还敢打老子？"
- 求饶时："我再也不在 `{name}` 面前嘚瑟了..."
- 崩溃时："`{name}`救救我！不，谁也救不了我了..."
- 小偷猪偷钱："谢谢老板的 `{gold}` 金币，我去找 `{name}` 吃烧烤了~"
- Boss 出场："听说你最高也就 `{combo}` 连击？`{name}` 都比你能打！"

**语音包扩展**（`ShopConfig.voicePacks` 新增 3 种）：
- 老板腔："这个月的 KPI 你完成了吗？"
- 绿茶腔："哥哥你打人家好痛哦~但是人家不怪你呢~"
- 唐僧腔："打猪是不对的，不过你要是真想打，那我也没办法。"

## Phase 2：爽感升级

### 2.1 打击反馈增强（不改架构，纯体验优化）

**屏幕震动分级**：
- 普通命中：轻震（已有）
- 暴击/重锤/火箭爆炸：中震
- Boss 弱点打破/完美防线通关：重震

**连击特效升级**：
- 10 combo：武器轨迹变色
- 30 combo：屏幕边缘红色脉动
- 50 combo：全屏慢动作 0.5 秒 + "UNSTOPPABLE" 大字浮现
- 100 combo：全场猪定身 1 秒 + "GODLIKE"

**猪的物理反应**：
- 击飞轨迹带随机旋转（不全是直线飞出）
- 猪撞到屏幕边缘弹一下再飞走
- 多只猪同时被击飞→互相碰撞

### 2.2 连击"无双时刻"

- 连击达到 50 后进入怒气模式 5 秒
  - 所有伤害 x1.5
  - 所有猪移速减半
  - 背景变红、音效加速

### 2.3 猪潮来袭（主玩法内随机事件）

主玩法中每隔 5 关触发一次"猪潮来袭"特殊波：
- 15 秒，同时涌出 15+ 只猪
- 每只猪 hp=1，一碰就飞
- 猪潮中金币掉落 x2
- 屏幕顶部显示 "🐷🐷🐷 猪潮来袭！🐷🐷🐷"
- 15 秒后恢复正常
- **不单独成副本，融入主线节奏**，给玩家一个爽感冲刺

## Phase 4：副本系统（全新玩法模式）

### 4.1 设计原则

副本是主线的补充，不是替代。提供**不同胜利条件、不同敌人配置、限时开放**。

### 4.2 副本类型（3 个，和主玩法本质不同）

#### 副本 A：消消揍（每日 5 次免费）★ 留存核心

**被微信小游戏验证过的高留存玩法**（消消乐次日留存 30%+），和暴打结合：

**棋盘**：
- 8×8 方格棋盘
- 方块是武器图标（🧹🔨🪰⚡🩴🚀🔫💩），8 种对应 8 把武器
- 交换相邻两个，连成 3+ 同色即消除

**消除 → 揍猪**：
- 每次消除，对应武器打在猪身上（播放对应武器的击打动画 + 惨叫台词）
- 消除数量 = 对猪的伤害值
- 3 消 → 普通一击（对应武器基础伤害）
- 4 消 → 暴击（伤害 x2 + 武器特效）
- 5 消 / T 型 / L 型 → 大招（伤害 x3 + 屏幕震动）
- 连锁 combo → 多种武器连招动画，触发"无双连揍"全屏特效
- 方块图标直接复用武器系统的 emoji，零额外美术成本

**关卡规则**：
- 每关 30 步，上方站一只小猪（从欠揍名单中随机选名）
- 步数用完前把猪揍死 → 胜利，金币奖励
- 步数用完猪还没死 → 猪嘲讽"就这？`{name}` 都比你消得快！"
- 小猪血量和当前主线关卡进度挂钩
- 每 5 关出现 Boss 猪（血量翻倍）

**和传统消消乐的区别**：
- 消除不是为了分数，是为了揍猪
- 每关有明确的"敌人"（带名字的猪），有揍它的动机
- 猪在挨揍时实时反馈（表情变化 + 台词）
- 步数耗尽时猪的嘲讽让玩家想再试一次

**实现要点**：
- `src/systems/Match3Engine.js`：棋盘逻辑、消除判定、下落填充、combo 检测
- `src/scenes/Match3Scene.js`：棋盘 Canvas 渲染 + 上方猪 + HUD
- 猪的反应复用 DialogBubble + Pig 动画

#### 副本 B：无限模式（每日 1 次免费 + 广告 +1 次）
- 没有时间限制，猪越来越快、越来越多，打到防线被突破
- 排行榜：最高击杀数 / 最高连击
- 奖励：按击杀数阶梯给金币

#### 副本 C：悬赏令（每日刷新 3 个）
- 从玩家"欠揍名单"中随机选 3 个名字生成悬赏猪
- 悬赏猪血更厚、有特殊技能、带赏金标记
- 击败后 x3 金币 + 该名字被揍次数 +1
- 可通过分享链接接收群友的"通缉"（query 参数携带 targetName）
- 被通缉的目标名字自动加入接收方欠揍名单

### 4.3 副本入口
- 主菜单新增"副本"按钮，用通缉令/公告板视觉风格
- 每个副本显示今日剩余次数 + 奖励预览

## Phase 5：社交裂变

### 5.1 分享战绩卡片

**现状**：`SettlementScene._share()` 纯文字分享
**改进**：Canvas 绘制战绩卡片 → `wx.shareFileMessage` 转发到群

卡片布局：
```
┌──────────────────────────┐
│  👤 玩家头像  玩家昵称      │
│                          │
│    🐷 [被揍猪大头贴]       │
│    猪的表情（肿包/求饶）    │
│    名字：XXX               │
│                          │
│  🎯 第 15 关  💰 830     │
│  💢 47 击杀  🔥 62 连击 │
│                          │
│  扫码一起揍 →  [二维码]    │
└──────────────────────────┘
```

**分享文案**："我把 XXX 揍飞了 47 次！你也来试试？"

**裂变机制**：卡片上的二维码 + `query` 参数携带邀请者 ID，新用户进入时自动加入邀请者的"揍友圈"（见 5.3）。

### 5.2 排行榜增强

**现状**：`RankManager` 上传 maxCombo / totalKills
**新增维度**：
- 本周揍飞最多（周榜，周一清零）
- 最高连击
- 最快 Boss 击杀
- 副本最高分
- "被揍最多"榜（欠揍名单中哪个名字被揍次数最高——娱乐向）

### 5.3 "揍友圈"通缉社交

**概念**：不是导入微信好友，而是建立游戏内社交关系。玩家之间通过分享链接互相关联，形成"揍友圈"。

**流程**：
1. 玩家 A 通关 → 分享战绩卡片到微信群
2. 玩家 B 点击卡片进入游戏 → 通过 query 参数自动和 A 建立"揍友"关系
3. A 和 B 的欠揍名单互相可见（可选开启/关闭）
4. 好友之间可以发送"通缉令"：A 选择 B 名单中的某个名字，悬赏金币，生成通缉卡片
5. B 收到通缉后，在悬赏副本中击败该名字的猪可获得悬赏金
6. 悬赏完成后双方都获得奖励

**隐私保护**：
- 欠揍名单默认仅自己可见
- 开启"和揍友共享名单"后揍友才能看到
- 通缉令不暴露发件人的名单，只暴露目标名字

### 5.4 每日挑战

- 每天随机生成一个挑战任务：
  - "用机关枪达成 30 连击"
  - "不被小偷猪偷走金币通关"
  - "一局击败 5 只 Boss"
- 完成奖励双倍金币
- 分享挑战结果到群

## 和同类游戏的核心差异

1. **敌人有名字**：不是"普通猪/快速猪"——是玩家亲手输入的名字，揍起来有动机
2. **猪会说话**：会嘲讽、求饶、崩溃，名字和玩家数据嵌入台词
3. **分享即裂变**：战绩卡片不是炫耀分数，是"我把 XXX 揍飞了"——接收方产生好奇心
4. **双重核心玩法**：主线"暴打"（动作解压，含猪潮来袭爽关）+ 副本"消消揍"（策略消除），玩法本质不同，互相导流
5. **揍友圈社交**：不依赖微信好友导入，通过分享链接建立游戏内社交关系
6. **帮派叙事**：每 10 关换一个世界，有期待感
